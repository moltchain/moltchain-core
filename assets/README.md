# Moltchain Brand Assets

This directory contains brand assets for Moltchain.

## Files

- `banner.png` - Main banner for README and social media (1500x500)
- `logo.png` - Primary logo (square, various sizes)
- `icon.png` - App icon/favicon

## Usage

These assets are available for use by the Moltchain community. Please follow our brand guidelines:

- Maintain aspect ratios
- Don't modify colors
- Provide adequate spacing
- Don't combine with other brands without permission

For full brand guidelines, visit: https://moltchain.xyz/brand

You can download the full media kit from: https://moltchain.xyz/media-kit
